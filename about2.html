chat_bubble_outline ID selector
- ID selectors have a higher specificity than attribute selectors
- This category selects element with a specific id attribute
- Start with #
Example

#box {
background-color: #ECECEC;
border-radius: 10px;
}


chat_bubble Grouping Selector
- This category make it specify the same properties and rules for more than one element at the same time
- Separate the selector names with a comma
Example

h1, h2, h3 {
background-color: #F5F5F5;
text-align: center;
}
h1 {
text-align: left;
padding: 10px;
}
h2, h3 {
padding: 25px;
}
