Let's define specificity

Element Selector keyboard_arrow_right Class Selector keyboard_arrow_right Id Selector

Element Selector : specificity = 1
Class Selector : specificity = 10
Id Selector : specificity = 100
Example

A : box { ... }
B : .center { ... }
C : #head { ... }
D : h1, h2, .box { ... }
E : #head h1{ ... }
The specificity of A is 1 (one element)
The specificity of B is 10 (one class reference)
The specificity of C is 100 (one ID reference)
The specificity of D is 12 (one class reference + two element)
The specificity of E is 101 (one ID reference + one element)
